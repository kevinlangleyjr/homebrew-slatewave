# homebrew-slatewave

The Homebrew tap for the [Slatewave](https://getslatewave.com) family.

> _Slate below, teal above._

</div>

---

## Install

```sh
brew tap kevinlangleyjr/slatewave
brew install slatewave
```

That installs the [`slatewave`](https://github.com/kevinlangleyjr/slatewave-cli) CLI — the family installer for every theme in the Slatewave family.

```sh
slatewave list                  # every theme, with ● / ○ install markers
slatewave install <theme>       # install + activate one theme
slatewave uninstall <theme>     # reverse files, restore backups, remove appended lines
```

## Upgrade

```sh
brew update
brew upgrade slatewave
```

## What's in this tap

- [`slatewave.rb`](slatewave.rb) — formula for the `slatewave` CLI

The formula is generated by [GoReleaser](https://goreleaser.com) on each [`slatewave-cli`](https://github.com/kevinlangleyjr/slatewave-cli) release; do not edit it by hand.

## Slatewave family

One palette. Every tool. See [getslatewave.com](https://getslatewave.com) for the full list.
